# Packages

The public index for Numo packages.

## Contributing

Submit a pull request, with your username in a folder, with your project also in the folder with your username, and then a file named `meta.json`, with inside, something like this:
```jsonc
{
  "name": "my-numo-pkg",
  "version": "1.0.0",
  "repository": "https://github.com/[...]/[...].git" // any git repository, ending in .git
}
```

And that's all! We'll review your codebase as well if everything is okay.
